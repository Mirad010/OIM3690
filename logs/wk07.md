# Week 7 Log

This week I built the three class exercises.

playlist.html — a list of songs I listen to. It uses one array of song objects
and a for loop that creates one <li> per song. This render-from-data pattern
was new for me but it makes sense: data in one place, loop builds the page.

post-message.html — an async function that sends a POST request to the class
server with my X-Token header. I added an input and a button so I can send any
message.
fetch-fun.html — I used two free APIs, Dog CEO for random dog photos and
Advice Slip for random advice. The tricky part was finding the right field in
the JSON: for the dog it is data.message, for advice it is data.slip.advice.
The Surprise me button picks one of the two APIs randomly with Math.random().
